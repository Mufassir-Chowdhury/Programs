We can also access random places of a file. We do that by using lseek() which can modify the file pointer.



off_t is and int-like type holding the largest acceptable file size. The offset is the length of byte to move the file pointer. The whence tells where to start counting the offset.

- SEEK_SET move the file pointer to the offset location.
- SEEK_CUR move the file pointer relative to current position.
- SEEK_END move the file from the relative to the end.

This function returns the final position or -1 if something goes wrong.




This shows that you can seek to a position further away from the end.
