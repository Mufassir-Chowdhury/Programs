A variable number of parameters is handled in C at two levels:
- Higher level: At the language's syntax level. Which tells the compiler about the speccial feature.
- Lower level: At the execution environment level. Which physically accesses the parameters.

Three dots named ellipsis is used to indicate the variable part.

Rules:

1. The ellipsis must the the last element of a prameter's list.

2. The ellipsis mustn't be the only element of a parameter's list.

3. The ellipsis must be used only once.

- A function with variable number of arguments is called a variadic function.
- All parameters from the fixed part is called the fixed paramteres.
- All parameters hiding in the ellipsis is called the variable part .

For example, the syntax for printf() function is,

extern int printf(const char *__restrict --format, ...);
